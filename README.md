# TPlayer 使用说明

## 1. 快速开始

### 1.1 安装

1. **下载安装包**：从 [GitHub 仓库](https://github.com/ChrisHcn1/TPlayer) 下载对应平台的安装包
2. **安装应用**：
   - Windows：双击 `.exe` 安装包，按照提示完成安装
   - macOS：双击 `.dmg` 文件，将 TPlayer 拖动到 Applications 文件夹
   - Linux：双击 `.AppImage` 文件，或使用包管理器安装

### 1.2 首次运行

1. **启动应用**：点击桌面上的 TPlayer 图标
2. **扫描音乐**：点击界面上方的"扫描目录"按钮，选择包含音乐文件的文件夹
3. **开始播放**：在播放列表中选择歌曲，点击播放按钮

## 2. 界面介绍

## 3. 基本操作

### 3.1 扫描音乐

1. 点击顶部工具栏中的"扫描目录"按钮
2. 在弹出的对话框中选择包含音乐文件的文件夹
3. 点击"确定"开始扫描
4. 扫描完成后，歌曲会显示在播放列表中

### 3.2 播放歌曲

1. 在播放列表中找到要播放的歌曲
2. 点击歌曲名称开始播放
3. 或右键点击歌曲，选择"播放"

### 3.3 控制播放

- **播放/暂停**：点击播放控制栏中的播放/暂停按钮
- **上一首**：点击播放控制栏中的上一首按钮
- **下一首**：点击播放控制栏中的下一首按钮
- **调整音量**：拖动播放控制栏中的音量滑块
- **调整播放进度**：拖动播放控制栏中的进度条

### 3.4 切换播放模式

点击播放控制栏中的播放模式按钮，循环切换：
- **顺序播放**：按列表顺序播放
- **随机播放**：随机播放列表中的歌曲
- **循环播放**：循环播放列表中的歌曲

## 4. 高级功能

### 4.1 歌单管理

#### 4.1.1 创建歌单

1. 点击左侧边栏中的"创建歌单"按钮
2. 输入歌单名称和描述
3. 点击"确定"创建歌单

#### 4.1.2 编辑歌单

1. 右键点击左侧边栏中的歌单
2. 选择"编辑歌单"
3. 修改歌单名称和描述
4. 点击"确定"保存更改

#### 4.1.3 删除歌单

1. 右键点击左侧边栏中的歌单
2. 选择"删除歌单"
3. 在确认对话框中点击"确定"

### 4.2 歌曲管理

#### 4.2.1 添加到歌单

1. 选择要添加的歌曲（可多选）
2. 右键点击选中的歌曲
3. 选择"添加到歌单"
4. 选择目标歌单

#### 4.2.2 添加到我喜欢

1. 点击歌曲右侧的心形按钮
2. 或右键点击歌曲，选择"添加到我喜欢"

#### 4.2.3 编辑歌曲信息

1. 右键点击歌曲
2. 选择"编辑"
3. 修改歌曲信息（标题、歌手、专辑等）
4. 点击"确定"保存更改

### 4.3 按歌手/专辑分组

1. 点击顶部工具栏中的"歌手"或"专辑"选项
2. 左侧会显示歌手或专辑列表
3. 点击左侧列表中的歌手或专辑，右侧会显示对应的歌曲

### 4.4 搜索歌曲

1. 在顶部工具栏的搜索框中输入关键词
2. 播放列表会实时显示匹配的歌曲

## 5. 设置选项

### 5.1 音频设置

- **音量**：调整默认音量
- **播放速度**：调整播放速度
- **交叉淡入淡出**：设置歌曲切换时的淡入淡出效果
- **转码质量**：设置音频转码的质量
- **预转码提前时间**：设置预转码的提前时间

### 5.2 界面设置

- **主题**：切换亮色或暗色主题
- **可视化**：启用或禁用音频频谱可视化
- **显示专辑封面**：显示或隐藏专辑封面

### 5.3 存储设置

- **播放进度记忆**：启用或禁用播放进度记忆
- **默认播放模式**：设置默认的播放模式

### 5.4 FFmpeg 设置

- **FFmpeg 路径**：指定 FFmpeg 可执行文件的路径
- **启用 FFmpeg**：启用或禁用 FFmpeg 解码

## 7. 支持的音频格式

TPlayer 支持以下音频格式：

- MP3
- FLAC
- WAV
- AAC
- OGG
- M4A
- APE
- DSD
- DTS
- WMA

对于不支持的格式，TPlayer 会自动使用 FFmpeg 转码为 FLAC 格式播放。

## 8. FFmpeg 配置

### 8.1 安装 FFmpeg

1. **Windows**：
   - 从 [FFmpeg 官网](https://ffmpeg.org/download.html) 下载压缩包
   - 解压到任意目录
   - 将 `bin` 目录添加到系统环境变量 `PATH`

2. **macOS**：
   - 使用 Homebrew 安装：`brew install ffmpeg`

3. **Linux**：
   - 使用包管理器安装：`apt install ffmpeg`（Ubuntu/Debian）或 `yum install ffmpeg`（CentOS/RHEL）

### 8.2 验证 FFmpeg 安装

1. 打开命令行终端
2. 输入 `ffmpeg -version`
3. 如果显示 FFmpeg 版本信息，则安装成功

## 9. 常见问题

### 9.1 音频播放问题

**问题**：某些音频格式无法播放
**解决方案**：确保已正确安装 FFmpeg 并配置环境变量

**问题**：播放卡顿
**解决方案**：启用转码功能，调整转码质量

**问题**：没有声音
**解决方案**：检查系统音量和 TPlayer 音量设置

### 9.2 界面问题

**问题**：列表滚动不流畅
**解决方案**：确保启用了虚拟滚动，检查系统性能

**问题**：主题切换无效果
**解决方案**：重启应用后再试

**问题**：专辑封面不显示
**解决方案**：确保音频文件包含专辑封面信息，或在编辑歌曲信息时添加专辑封面

### 9.3 其他问题

**问题**：扫描目录无反应
**解决方案**：检查目录权限，确保目录中包含音频文件

**问题**：歌单保存失败
**解决方案**：检查存储空间，确保有足够的权限

**问题**：播放进度不保存
**解决方案**：在设置中启用播放进度记忆

## 10. 故障排除

### 10.1 日志查看

- **Windows**：日志文件位于 `%APPDATA%\TPlayer\logs`
- **macOS**：日志文件位于 `~/Library/Logs/TPlayer`
- **Linux**：日志文件位于 `~/.config/TPlayer/logs`

### 10.2 重置设置

1. 关闭 TPlayer
2. 删除配置文件：
   - **Windows**：`%APPDATA%\TPlayer\config.json`
   - **macOS**：`~/Library/Application Support/TPlayer/config.json`
   - **Linux**：`~/.config/TPlayer/config.json`
3. 重新启动 TPlayer
